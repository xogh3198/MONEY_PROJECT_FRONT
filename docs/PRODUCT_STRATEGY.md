# InvestBoard 제품·수익화 전략

마지막 검증일: 2026-07-22

## 1. 목표

InvestBoard의 목표는 단기 조회수 극대화가 아니라, 한국 개인 투자자가 반복 방문할 이유가 있는 신뢰도 높은 금융 정보 제품을 만들고 지속 가능한 양(+)의 공헌이익을 내는 것이다.

핵심 선순환은 다음과 같다.

> 시의성 있는 신호 발견 → 고유한 해설 콘텐츠 → 숏폼 유입 → 근거와 도구가 있는 사이트 → 재방문·회원 전환 → 광고·제휴·구독·B2B 수익 → 더 나은 데이터와 콘텐츠

## 2. 현재 자산과 빈틈

### 이미 있는 자산

- 네이버 검색 API, RSS, 네이버 랭킹 기반 뉴스 수집
- 코스피·코스닥·환율·S&P 500·비트코인·금 시장지표
- AI 시장 분석, 포럼, 배당 포트폴리오, 계산 도구
- Vercel Analytics와 검색엔진용 메타데이터

### 수익화를 막는 빈틈

- 뉴스 제목을 보여주는 것 외에 InvestBoard만의 원문 해설이 적다.
- 콘텐츠 유입과 사이트 행동을 연결하는 캠페인 식별자가 없다.
- 수집 데이터의 금융 관련성, 시의성, 관심도를 콘텐츠 후보로 점수화하지 않는다.
- 반복 방문을 만드는 관심종목·알림·뉴스레터 루프가 약하다.
- 광고를 붙이기 전에 필요한 편집 정책, 제작 과정 공개, 개인정보·쿠키 고지가 부족하다.

## 3. 우선 고객

첫 고객은 모든 투자자가 아니라 다음 사용자로 좁힌다.

- 20~40대 한국 개인 투자자
- 출퇴근 중 1분 안에 중요한 시장 변화를 이해하고 싶은 사람
- 개별 종목 매수 지시보다 “무슨 일이 왜 중요하고 내 자산에 어떤 변수가 되는지”를 원하는 사람
- 짧은 콘텐츠에서 시작해 지표, 근거, 계산기까지 확인하려는 사람

핵심 사용자 과업은 “오늘 내 돈에 영향을 줄 변화를 빠르게 이해하고, 더 깊게 확인할 곳을 찾는 것”이다.

## 4. 포지셔닝

InvestBoard는 속보 복제 서비스가 아니라 **데이터 기반 금융 이슈 설명서**가 된다.

모든 핵심 콘텐츠는 최소한 아래 중 하나의 고유 가치를 더해야 한다.

- 여러 신호를 연결한 원인·영향 설명
- 숫자와 과거 구간을 사용한 비교
- 개인 투자자 관점의 체크리스트
- 계산기, 차트, 캘린더 같은 실행 가능한 도구
- 사후 결과 추적과 기존 설명의 정확도 공개

## 5. 수익 구조

### 1단계: 유입과 신뢰

- YouTube Shorts, Instagram Reels용 원본 해설 콘텐츠
- 검색 가능한 원본 브리핑/설명 페이지
- 무료 시장 요약과 관심 주제 알림

이 단계의 목표는 광고 매출이 아니라 재방문 사용자와 검증된 콘텐츠 포맷을 만드는 것이다.

### 2단계: 낮은 마찰의 수익

- 원본 콘텐츠가 충분한 페이지의 문맥형 광고
- 명확한 경제적 이해관계 표시가 있는 제휴 링크
- 브랜드 적합성과 편집 독립성을 지키는 스폰서 콘텐츠

광고는 트래픽의 바닥 수익이고 핵심 방어력은 아니다. 제휴는 사용자의 과업과 맞는 제도권 서비스만 검토하며 클릭이나 가입을 과도하게 유도하지 않는다.

### 3단계: 반복 수익

- 관심 주제·포트폴리오 기반 프리미엄 알림
- 광고 없는 환경과 고급 비교·백테스트·리포트
- 팀/크리에이터용 트렌드·콘텐츠 데이터 피드

유료 종목 추천이나 매매 시점 제공은 규제 검토와 별도 승인이 끝나기 전에는 제품 범위에 넣지 않는다.

### 4단계: B2B

- 정규화한 금융 뉴스 신호 API
- 콘텐츠 팀용 트렌드 탐지 및 성과 회고 도구
- 제도권 파트너 대상 공동 리서치·스폰서십

## 6. 핵심 지표

### North Star

`주간 유효 재방문 사용자 수`

유효 사용자는 2개 이상의 고유 콘텐츠/도구를 보거나 관심 주제 저장 등 핵심 행동을 완료한 사용자다.

### 획득과 콘텐츠

- 콘텐츠별 3초 유지율, 평균 시청 비율, 완주율
- 숏폼 → 사이트 CTR (`utm_source`, `utm_campaign`, `utm_content`)
- 콘텐츠 후보 1건당 제작 시간과 외부 API 비용
- 게시 24시간/7일 후 조회수 대비 사이트 유입

### 제품과 수익

- 방문 → 회원/관심 주제 저장 전환율
- 7일·30일 재방문율
- 광고 페이지 RPM과 광고 없는 페이지의 전환율
- 제휴 CTR·승인 전환율·취소율
- 유료 전환율, ARPU, 해지율
- 채널별 매출에서 변동비를 뺀 공헌이익

## 7. 우선순위

| 순서 | 실험 | 이유 | 승인 게이트 |
|---|---|---|---|
| 1 | 콘텐츠 기회 탐지·쇼츠 초안 스튜디오 | 현재 데이터를 바로 재사용하며 수요를 가장 싸게 검증 | 외부 게시 전 승인 |
| 2 | 10~20개 원본 금융 설명 페이지 | 검색·AdSense·사이트 신뢰의 공통 기반 | 게시 전 사실/권리 검수 |
| 3 | 수동 숏폼 10개 포맷 테스트 | 자동화 전에 훅·길이·CTA의 실제 성과 확인 | 계정 생성·게시 승인 |
| 4 | 관심 주제 저장·일일 브리핑 | 재방문과 회원 전환 검증 | 메시지 발송 동의 |
| 5 | 광고 적합성·소수 슬롯 실험 | 충분한 원본 콘텐츠 이후 바닥 수익 확인 | AdSense 약관·계정 승인 |
| 6 | 프리미엄 알림 수요 조사 | 반복 매출 가능성 검증 | 결제·규제 검토 |

## 8. 90일 실행안

### 0~30일: 계측과 포맷 검증

- 콘텐츠 스튜디오 MVP 구축
- 종료된 AI 모델 교체와 생성 결과 구조화
- 원본 콘텐츠 제작/검수 정책 공개 준비
- 3개 포맷 × 각 3~4개 숏폼을 수동 검수해 게시
- 모든 링크에 캠페인 태그 적용

### 31~60일: 사이트 가치 강화

- 성과가 좋은 주제의 원본 설명 페이지 템플릿 구축
- 관심 주제 저장과 일일 브리핑 실험
- 영상 성과와 사이트 행동을 같은 실험 ID로 연결
- 저성과 포맷 중단, 상위 포맷에 집중

### 61~90일: 수익 검증

- 원본 콘텐츠, 개인정보·쿠키, 편집 정책을 갖춘 뒤 AdSense 심사 검토
- 사용자 과업과 맞는 제휴 1개만 제한적으로 검증
- 광고 대비 프리미엄 기능의 지불 의사 조사
- 채널별 공헌이익 기준으로 다음 분기 투자 결정

## 9. 정책에서 나온 제품 원칙

- YouTube 광고 수익 기준은 일반적으로 구독자 1,000명과 최근 90일 유효 Shorts 조회수 1,000만 회 또는 최근 12개월 공개 영상 시청시간 4,000시간이다. 광고로 산 Shorts 조회수는 기준에 포함되지 않는다.
- 반복 템플릿, 대량 생산, 기사 낭독처럼 원작성과 해설 가치가 약한 콘텐츠는 채널 수익화에 불리하다.
- 현실적으로 보이는 합성 영상·음성은 플랫폼 요구에 따라 공개하며, 금융 주제에는 더 눈에 띄는 라벨이 붙을 수 있다.
- AdSense는 고유하고 흥미로운 게시자 콘텐츠를 요구하며, 자동 생성·복제 콘텐츠나 광고가 콘텐츠보다 많은 화면에 불리하다.
- 광고 클릭을 요청하거나 자동·반복 클릭을 만드는 행위는 금지한다.
- 금융 콘텐츠는 Google이 높은 신뢰성을 요구하는 YMYL 영역이므로 작성자, 출처, 제작 방식, 검토자를 분명히 한다.
